Aim

To design a lead compensator for the given transfer function using Scilab toolbox and analyze system performance using Bode plots and step response.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

A compensator is a device used in control systems to modify system performance to meet desired specifications such as stability, speed, and accuracy.

Given Transfer Function:

G(s) = K / (s² (s + 1))

Adding a pole shifts the root locus to the right, reducing stability and increasing settling time. Adding a zero shifts the root locus to the left, improving stability and reducing settling time.

Types of Compensators:

  • Lag Compensator
  • Lead Compensator
  • Lag-Lead Compensator

In this experiment, a lead compensator is used to improve phase margin and transient response.

Procedure

  1. Open Scilab software.
  2. Define Laplace variable using s = %s.
  3. Define transfer function using syslin().
  4. Plot Bode plot of uncompensated system.
  5. Find gain margin and phase margin.
  6. Enter desired phase margin and epsilon.
  7. Calculate required phase lead.
  8. Compute alpha and time constant T.
  9. Design lead compensator:
    Gc = (s + 1/T) / (s + 1/(αT))
  10. Form compensated system.
  11. Plot Bode plot and step response.
  12. Compare compensated and uncompensated systems.

Result

The lead compensator was successfully designed using Scilab. The compensated system showed improved phase margin, better stability, and faster response compared to the uncompensated system. The design satisfies the required specifications.

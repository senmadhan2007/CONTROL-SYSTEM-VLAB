Aim

To plot the Nyquist diagram of a given transfer function using Scilab and analyze the stability of the system.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

The Nyquist plot is a graphical representation of the frequency response of a system in the complex plane. It is used to determine the stability of a closed-loop control system using the Nyquist stability criterion.

The given transfer function is:

G(s) = 1 / (s³ + 3s² + 6s + 1)

By substituting s = jω, the frequency response G(jω) is obtained.

The Nyquist plot shows how the complex value of the transfer function varies with frequency.

  • Encirclement of the point (-1, 0) determines system stability.
  • No encirclement → Stable system (for open-loop stable systems).
  • Encirclement → Possible instability.

Nyquist plots are widely used in control system analysis for stability and performance evaluation.

Procedure

  1. Open Scilab software.
  2. Define the Laplace variable using s = poly(0, 's').
  3. Define the transfer function:
    Sys = syslin('c', 1/(s^3 + 3*s^2 + 6*s + 1))
  4. Define the time range (optional): t = 0:0.05:30.
  5. Use the Nyquist command:
    nyquist(Sys)
  6. Observe the Nyquist plot generated.
  7. Analyze encirclements of (-1,0) point to determine stability.

Result

The Nyquist plot of the given transfer function was successfully obtained using Scilab. The plot was analyzed to determine the stability of the system based on the Nyquist stability criterion. The system behavior was observed in the frequency domain.

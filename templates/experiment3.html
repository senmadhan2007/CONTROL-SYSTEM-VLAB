Aim

To study and analyze the time-domain response of a second order control system and determine its performance parameters such as peak time, overshoot, rise time, delay time, and settling time using Scilab simulation.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

A second order control system is represented by a differential equation containing two energy storage elements. Its standard transfer function is given by:

G(s) = ωn2 / (s2 + 2ζωns + ωn2)

The time-domain response of such a system provides insight into its dynamic behavior, stability, speed of response, and damping characteristics.

Procedure

  1. Select Experiment 3 from the experiment list.
  2. Observe the reference Scilab program provided.
  3. Type the program in the editor exactly.
  4. Modify only the system parameters if instructed.
  5. Click Execute to run the simulation.
  6. Observe the response and note the results.

Result

The time-domain response of the second order system was successfully simulated using Scilab. Performance parameters such as peak time, rise time, settling time, and overshoot were determined and analyzed.

Aim

To study and analyze the frequency response of a linear time-invariant system using Bode plot and determine system characteristics such as gain margin and phase margin using Scilab.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

Frequency response analysis studies how a system responds to inputs of varying frequencies.

A Bode plot consists of:

  • Magnitude Plot: Gain (in dB) vs frequency
  • Phase Plot: Phase (in degrees) vs frequency

The transfer function is evaluated by replacing s with jω:

G(s) → G(jω)

Magnitude is calculated as:

20 log₁₀ |G(jω)|

Stability is determined using gain margin and phase margin.

Procedure

  1. Open Scilab software.
  2. Define Laplace variable using poly().
  3. Define transfer function using syslin().
  4. Use bode() to plot the Bode diagram.
  5. Analyze magnitude and phase plots.
  6. Determine system stability using GM and PM.

Result

The Bode plot of the system was obtained successfully. The magnitude and phase characteristics were analyzed, and system stability was evaluated using gain margin and phase margin.

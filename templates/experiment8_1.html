Experiment 8 – Frequency Response Analysis Using Polar Plot

Reference Program

T = 10;
L = 100;

omega = logspace(-1, 2, 1000);
s = %i * omega;

den = s .* (T * s + 1);
num = exp(-s * L);

repf = num ./ den;

rad = abs(repf);
theta = atan(imag(repf), real(repf));

polarplot(theta, rad, style=5);

Aim

To study and analyze the stability of a linear time-invariant system using the Root Locus technique and determine system behavior for different values of gain K using Scilab.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

Root Locus is a graphical method used to study how the roots of the characteristic equation of a control system change with variation in system gain (K).

The characteristic equation is given by:

1 + G(s)H(s) = 0

where G(s)H(s) is the open-loop transfer function.

The root locus shows the movement of system poles in the s-plane as K varies from 0 to infinity.

Stability conditions:

  • If all closed-loop poles lie in the left half of the s-plane, the system is Stable.
  • If any pole moves to the right half of the s-plane, the system is Unstable.
  • If poles lie on the imaginary axis, the system is Marginally Stable.

Root locus helps in selecting the appropriate gain K to achieve desired stability and performance.

Procedure

  1. Open Scilab software.
  2. Define the Laplace variable using poly().
  3. Define the transfer function using syslin().
  4. Use evans() to plot the root locus.
  5. Observe pole movement in the s-plane.
  6. Determine stability based on pole location.

Result

The root locus plot of the given system was successfully obtained using Scilab. The variation of pole locations with respect to gain K was analyzed, and the stability of the system was determined.

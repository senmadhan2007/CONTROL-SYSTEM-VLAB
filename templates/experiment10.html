Aim

To design a PID controller for first-order and second-order systems and analyze system performance to satisfy given specifications such as overshoot, rise time, settling time, and steady-state error.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Control System Toolbox
  • Internet Browser
  • Virtual Control System Laboratory

Theory

A PID (Proportional–Integral–Derivative) controller is used to improve the performance of control systems by minimizing error and improving stability.

Controller Equation:

C(s) = Kp + Ki/s + Kd·s

Where Kp is proportional gain, Ki is integral gain, and Kd is derivative gain.

Given Systems:

1. G(s) = 3.6 / (s + 10)

2. G(s) = 5 / (s² + 6s + 5)

The PID controller helps achieve:

  • Reduced rise time
  • Reduced overshoot
  • Improved stability
  • Zero steady-state error

Controller parameters are tuned using trial and error to meet design specifications.

Procedure

  1. Open Scilab software.
  2. Define the Laplace variable using s = %s.
  3. Define the system transfer function using syslin().
  4. Plot the open-loop response using csim().
  5. Select initial values for Kp, Ki, and Kd.
  6. Define PID controller:
    Con = Kp + Ki/s + Kd*s
  7. Form closed-loop system:
    Sys_f = Con*Sys/(1 + Con*Sys)
  8. Plot closed-loop response.
  9. Adjust Kp, Ki, Kd until specifications are satisfied.
  10. Compare open-loop and closed-loop responses.

Result

The PID controller was successfully designed using Scilab. The system response showed improved performance with reduced overshoot, faster rise time, acceptable settling time, and zero steady-state error. The design met the required specifications.

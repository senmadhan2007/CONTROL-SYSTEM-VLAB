Aim

To analyze the frequency response of a given system using a polar plot and study its magnitude and phase characteristics.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

Frequency response analysis studies how a system behaves for different input frequencies.

The transfer function of the system is:

G(s) = e-sL / [ s(Ts + 1) ]

By substituting s = jω, we obtain the frequency response:

G(jω)

Polar Plot

A polar plot represents the magnitude and phase of a system in the complex plane.

  • Magnitude: |G(jω)|
  • Phase: ∠G(jω) = tan-1(Imag/Real)

The exponential term introduces phase lag, which affects system stability.

Polar plots are useful in stability analysis and understanding system behavior.

Procedure

  1. Open Scilab software.
  2. Define system parameters T = 10 and L = 100.
  3. Generate frequency range using omega = logspace(-1, 2, 1000).
  4. Define complex variable s = %i * omega.
  5. Compute transfer function using numerator and denominator expressions.
  6. Calculate magnitude and phase, then plot using polarplot(theta, rad).

Result

The polar plot of the given system was obtained successfully. The magnitude and phase characteristics were analyzed. The system shows frequency-dependent behavior, and the delay introduces additional phase lag.

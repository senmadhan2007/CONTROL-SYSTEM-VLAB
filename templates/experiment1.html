Aim

To study and analyze the time-domain responses (step, impulse, ramp, and sinusoidal) of first order and second order control systems using simulation.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

Control systems are mathematical models used to regulate the behavior of dynamic systems. The time response of a system indicates how the system reacts to different inputs such as step, impulse, ramp, and sinusoidal signals.

First order systems are characterized by a single energy storage element, while second order systems contain two energy storage elements. Studying their responses helps in understanding system stability, speed of response, and steady-state behavior.

Procedure

  1. Select the required experiment from the list.
  2. Observe the reference Scilab program provided.
  3. Type the program in the given editor.
  4. Modify only the system parameters if required.
  5. Click the Execute button to view the output.
  6. Analyze the response obtained.

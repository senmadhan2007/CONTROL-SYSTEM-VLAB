Aim

To study and analyze the stability of linear time-invariant systems using pole-zero plots and determine the system stability based on the location of poles in the s-plane using Scilab.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

The stability of a linear time-invariant (LTI) system can be determined from the location of its poles in the complex s-plane. The transfer function of a system is given by:

G(s) = N(s) / D(s)

where N(s) represents the numerator polynomial (zeros) and D(s) represents the denominator polynomial (poles).

The system stability conditions are:

  • If all poles lie in the left half of the s-plane (negative real part), the system is Stable.
  • If any pole lies in the right half of the s-plane (positive real part), the system is Unstable.
  • If poles lie on the imaginary axis (real part zero), the system is Marginally Stable.

The pole-zero plot provides a graphical representation of poles and zeros and helps in analyzing system stability and dynamic characteristics.

Procedure

  1. Select the experiment “Stability Analysis using Pole-Zero Plot”.
  2. Observe the given system transfer function.
  3. Enter the transfer function in Scilab using syslin().
  4. Generate the pole-zero plot using plzr().
  5. Observe the pole locations in the s-plane.
  6. Determine whether the system is stable, unstable, or marginally stable.

Result

The pole-zero plot of the given linear system was successfully obtained using Scilab. Based on the location of poles in the s-plane, the stability of the system was analyzed and verified.

Aim

To determine the stability of a linear control system using the Routh Hurwitz Stability Criterion by constructing the Routh array from the characteristic equation of the system.

Apparatus Required

  • Computer / Laptop
  • Scilab Software
  • Internet Browser
  • Virtual Control System Laboratory

Theory

The Routh Hurwitz stability criterion is a mathematical technique used in control systems to determine the stability of a linear time-invariant system without solving the characteristic equation.

Consider the characteristic equation of a system:

ansn + an-1sn-1 + ... + a1s + a0 = 0

Using the coefficients of this equation, a Routh array is constructed. The stability of the system depends on the elements of the first column of the Routh array.

  • If all elements in the first column are positive and no sign changes occur, the system is stable.
  • If there are sign changes, the system is unstable.
  • The number of sign changes equals the number of roots in the right half of the s-plane.

Procedure

  1. Write the characteristic equation of the given control system.
  2. Arrange the coefficients in descending order of powers of s.
  3. Construct the first two rows of the Routh array using the coefficients.
  4. Calculate the remaining rows using the Routh Hurwitz formula.
  5. Observe the signs of the elements in the first column.
  6. Count the number of sign changes in the first column.
  7. If there are no sign changes, the system is stable.
  8. If sign changes occur, the system is unstable.
  9. Verify the result using Scilab simulation.

Result

The stability of the given linear control system was determined using the Routh Hurwitz criterion. By analyzing the sign changes in the first column of the Routh array, the system stability was successfully verified.
